# Sunshine - forecast weather application
---
**The application is made according to the course "Developing Android Apps by Google" at Udacity.com**

Data source: https://openweathermap.org

### Tech stack:

- **Java**
- **SQLite**
- **Loader**
- **REST API**
- **Firebase JobDispatcher**
